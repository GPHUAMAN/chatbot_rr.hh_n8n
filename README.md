# 🤖 Agente de IA para RR.HH. - Chocolatech

## Descripción
Este proyecto implementa un agente de Inteligencia Artificial que responde automáticamente preguntas frecuentes del área de Recursos Humanos de la empresa ficticia Chocolatech a través de un chatbot de Telegram.

El objetivo es reducir el tiempo de atención a consultas repetitivas y mejorar la experiencia de los colaboradores.

## Funcionalidades
- Responde preguntas frecuentes de RR.HH.
- Integración con Telegram.
- Uso de un modelo de lenguaje (LLM).
- Respuestas basadas en documentación de la empresa.
- Conversación en lenguaje natural.

## Tecnologías utilizadas
- Python
- Telegram Bot API
- OpenAI API (o el modelo que utilices)
- LangChain (si aplica)
- Git y GitHub
